  • Subject headings were extracted from the set of records and written to a CSV file with corresponding title information from MARC field 245: http://timathom.github.io/sac/data/sac-wg-data-1.csv. A second version of the data was generated in which terms in the subject column were tokenized, stemmed, and lowercased, with stopwords removed: http://timathom.github.io/sac/data/sac-wg-data-tokenized-stemmed.csv
  • A co-occurence network of subject terms was extracted from the data using the Sci2 data visualization platform. This network shows which subject terms occur together in the same records.
  • Edges with a weight of 1 were removed from the network and isolated nodes were deleted.
  • The network was imported into the graph visualization platform Gephi and analyzed using the software's built-in algorithms. A new CSV file including Gephi's network statistics was exported: http://timathom.github.io/sac/data/sac-wg-gephi-analysis.csv
  • One of Gephi's algorithms permits community detection or clustering of nodes in a graph. Based on this analysis, the original terms were grouped accordingly. The HTML table, below, shows the results. A total of 212 groups was detected. Most of the groups are relatively sparse, with the majority of headings clustered into the last 20 or so groupings.
